the_minimum

Readability is not requared.

  • Focus on text. Maximum reading experience.
  • Adapt responsive web design. So that you don't have to worry about smartphones
  • Made with HTML5 and CSS3 (Not for fancy shiny things you can do with them)
  • Using LESS. Easier CSS editing.
  • Two different layouts. One for top page and the other for articles.
  • File structure is kept minimum so it'd be easy to custamize theme.
